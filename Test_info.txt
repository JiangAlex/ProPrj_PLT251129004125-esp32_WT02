參數,類型,說明
pvTaskCode,TaskFunction_t,任務函數的指標。 這是任務的實際程式碼。該函數必須具有 void taskFunction(void *pvParameters) 的簽名。
pcName,const char *,任務名稱。 用於調試和 FreeRTOS 運行時分析。
usStackDepth,uint16_t,任務堆疊大小。 以 bytes 為單位。對於 ESP32 上的 C++ 和 Arduino 框架，通常需要 2048 bytes 以上。LVGL 任務可能需要 4096∼8192 bytes。
pvParameters,void *,傳遞給任務函數的參數。 如果不需要參數，則設為 NULL。
uxPriority,UBaseType_t,任務優先級。 數字越大，優先級越高 (0 是最低)。LVGL 任務通常設為較高優先級。
pxCreatedTask,TaskHandle_t *,任務句柄指標。 儲存創建的任務的句柄，用於後續的控制（例如刪除任務）。如果不需要，則設為 NULL。
